# Triptych — IDE-Style Three-Panel Jekyll Theme

## Overview

**Triptych** is a Jekyll theme inspired by the classic  
**three-panel layout of modern IDEs** such as VS Code, IntelliJ, and Eclipse.

The layout is optimized for documentation-heavy and text-centric websites:

- **Left panel**: Global navigation
- **Center panel**: Content list
- **Right panel**: Content viewer

It is designed for developers, researchers, and writers who treat their website as a long-term working space rather than a feed-style blog.

---

## Layout Structure

```
┌─────────────┬─────────────────┬─────────────────────┐
│ Navigation  │ Content List    │ Viewer              │
│ (Sidebar)   │ (Posts / Items) │ (Markdown / HTML)   │
└─────────────┴─────────────────┴─────────────────────┘
```

### 1. Navigation Panel (Left)
- Categories and tags
- Static pages (About, CV, Contact)
- Collapsible tree-style navigation

### 2. Content List Panel (Center)
- Post or document listings
- Date, tags, and short summaries
- Optimized for fast scanning and navigation

### 3. Viewer Panel (Right)
- Markdown-first rendering
- Optimized for long-form reading
- Friendly to code blocks, tables, and math

## Use Cases

- Technical blogs (Linux, Embedded, Compilers, AI)

## Recommended Content Structure

```text
/
├── _posts/        # Technical articles and essays
├── _projects/     # Project documentation
├── _research/     # Research notes and papers
├── _teaching/     # Course materials
├── _pages/        # About, CV, Contact
```

---

## Design Philosophy

> Writing is not disposable content —  
> it is accumulated work.

This theme is not built for social-media-style consumption.  
It is designed as a **persistent workspace**, much like an IDE.
