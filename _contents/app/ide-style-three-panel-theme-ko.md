# Triptych — IDE 스타일 3패널 Jekyll 테마

## 개요

**Triptych**는 VS Code, IntelliJ, Eclipse 같은 현대 IDE의  
**3패널 레이아웃**에서 영감을 받은 Jekyll 테마입니다.

이 레이아웃은 문서 중심·텍스트 중심 웹사이트에 최적화되어 있습니다.

- **좌측 패널**: 전역 내비게이션
- **중앙 패널**: 콘텐츠 목록
- **우측 패널**: 콘텐츠 뷰어

피드형 블로그보다 장기 작업 공간으로 웹사이트를 활용하는 개발자, 연구자, 작성자를 위해 설계되었습니다.

---

## 레이아웃 구조

```
┌─────────────┬─────────────────┬─────────────────────┐
│ Navigation  │ Content List    │ Viewer              │
│ (Sidebar)   │ (Posts / Items) │ (Markdown / HTML)   │
└─────────────┴─────────────────┴─────────────────────┘
```

### 1. 내비게이션 패널(좌측)
- 카테고리/태그
- 정적 페이지(About, CV, Contact)
- 접기/펼치기 가능한 트리 내비게이션

### 2. 콘텐츠 목록 패널(중앙)
- 포스트/문서 목록
- 날짜, 태그, 요약 정보
- 빠른 훑어보기와 이동에 최적화

### 3. 뷰어 패널(우측)
- 마크다운 우선 렌더링
- 장문 읽기에 최적화
- 코드블록/표/수식 친화적 구성

## 핵심 기능

- 🧩 **IDE 유사 사용자 경험**  
  개발자·엔지니어에게 익숙한 구조

- 📝 **문서 중심 디자인**  
  기술 문서, 연구 노트에 적합

- 🎨 **미니멀·집중형 UI**  
  시각적 노이즈 최소화

- ⌨️ **키보드 친화 탐색**  
  효율적인 콘텐츠 이동 지원

- 🌙 **다크 모드 대응**  
  장시간 작업 환경에 적합

## 활용 사례

- 프로젝트 문서 아카이브
- 학술 포트폴리오(Research / Teaching / Projects)

---

## 권장 콘텐츠 구조

```text
/
├── _posts/        # 기술 아티클/에세이
├── _projects/     # 프로젝트 문서
├── _research/     # 연구 노트/논문
├── _teaching/     # 강의 자료
├── _pages/        # About, CV, Contact
```

---

## 디자인 철학

> 글쓰기는 소모성 콘텐츠가 아니라  
> 누적되는 작업물이다.

이 테마는 소셜 피드형 소비 모델을 목표로 하지 않습니다.  
IDE처럼 **지속 가능한 작업 공간**을 지향합니다.
